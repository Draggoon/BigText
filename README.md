# BigText
A small tool to create bigfont characters to bookmark C++ code.

## Note
- Not all characters are implemented yet
- The comment characters (//) are hard-coded yet

## Usage:
python3 PrintBigText.py "TEXT"

### outputs:
// ######### ######### ##,   ,## #########
// """###""" ##""""""   "#, ,#"  """###"""
//    ###    ##          "#,#"      ###   
//    ###    ########     ###       ###   
//    ###    ##"""""     ,#"#,      ###   
//    ###    ##,,,,,,   ,#" "#,     ###   
//    ###    ######### ##"   "##    ###
